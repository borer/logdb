# LogDB

[![Build Status](https://api.travis-ci.org/borer/logdb.svg?branch=master)](https://travis-ci.org/borer/logdb)

Log structured based database

To compile and run the the tests :

```sh
mvn clean
mvn package
```

To run the benchmark :

```sh
./runBenchmarks.sh
```
